## Summary

Implement the metrics panel and squad save on `/dashboard`. Three pure functions in `lib/metrics.ts`
(tested by Jest) compute total hourly cost, average seniority level, and unique skill coverage.
`MetricsPanel` calls `useSquad().members` and wraps all three calculations in a single `useMemo`.
`SaveSquadButton` manages a 4-state machine (idle → loading → success/error → idle) and calls a
`saveSquad` Server Action that POSTs to JSON Server's `/squads` collection. Both components are
rendered inside the existing right sidebar (`SquadPanel`), below the member list.

## Implementation Notes

### jest.config.ts

```typescript
const config: Config = {

```

`next/jest` handles TypeScript compilation via Babel — no `ts-jest` needed. `testEnvironment: 'node'` is appropriate since `lib/metrics.ts` has zero DOM dependencies.

### lib/metrics.ts

Three exported pure functions. No imports from React. Uses `Seniority` and `Developer` types from `lib/types.ts`.

Seniority scoring: `junior=1, mid=2, senior=3` → arithmetic mean → `Math.round()` → clamp [1,3] → map back to label. Returns `null` for empty squad.

Skill coverage: `members.flatMap(m => m.skills)` → `new Set()` → spread → `.sort()`.

### MetricsPanel.tsx

```typescript
'use client'
const metrics = useMemo(
  () => ({
    totalCost: calcTotalCost(members),
    avgSeniority: calcAvgSeniority(members),
    skillCoverage: calcSkillCoverage(members),
  }),
  [members]
)
```

Renders three metric cards: "$X/hr" (total cost), seniority label (or "—" when null), skill count with tooltip or list.

### SaveSquadButton.tsx

```typescript
'use client'
type SaveState = 'idle' | 'loading' | 'success' | 'error'
```

- `useRef<ReturnType<typeof setTimeout> | null>(null)` for timer cleanup
- `useEffect([members])`: if `saveState === 'success'`, cancel timer, reset to `'idle'`
- `useEffect return cleanup`: cancel pending timer on unmount
- `handleSave`: async, catches all errors, never throws to React

### app/(private)/dashboard/actions.ts

```typescript
'use server'
import type { Developer } from '@/lib/types'

export async function saveSquad(members: Developer[]) {
  if (members.length === 0) throw new Error('Cannot save empty squad')
  const res = await fetch('http://localhost:3001/squads', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ savedAt: new Date().toISOString(), members }),
  })
  if (!res.ok) throw new Error('Failed to save squad')
  return res.json()
}
```

### SquadPanel.tsx update
